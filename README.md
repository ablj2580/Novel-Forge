# 墨匣 Novel Forge Demo

一个给小说作者使用的 AI 辅助创意整理与正文写作原型。

## 当前实现

### 📚 作品管理
- 多本小说独立管理，支持书目颜色和写作状态
- 作品切换时自动同步设定资料库和关系图谱

### 📥 灵感收件箱
- 随手写入零散想法，记录 `pending` / `processing` / `done` 的 AI 状态
- **编辑功能**：支持修改标题、内容、目标模块、标签和归属作品
- 归档到设定资料库、回退到收件箱功能

### 🤖 AI 审阅
- 默认使用 DeepSeek 的 OpenAI 兼容 Chat Completions 接口
- 未配置 API Key 时使用本地规则模拟分类
- **自定义模块识别**：AI 分析可识别用户自定义的设定模块

### 🗄️ 设定资料库
- **模块管理**：人物、世界观、剧情线、金手指、伏笔、场景片段、道具、境界等
- **滚动条限制**：模块列表和资料列表均支持独立滚动
- **主角标签**：每本书可设置唯一主角，关系图谱默认以主角为中心展示
- 保留 AI 置信度，支持编辑、删除、查看关系

### 🔗 关系图谱
- **主角视图**：默认展示以主角为中心的关系图谱
- **关系类型管理**：支持新增、编辑、删除关系类型，自动使用最小可用ID
- **添加关联**：搜索设定资料建立关系，按当前作品过滤

### ✍️ 正文写作
- 章节编辑、自动保存、字数统计
- 右侧相关设定引用，支持模块过滤
- AI 分析设定：从正文中识别新设定元素

### 🔍 搜索功能
- 全局搜索设定资料，支持标题、内容、标签匹配
- 按当前作品过滤搜索结果

### 💾 数据管理
- 使用浏览器 `localStorage` 保存
- 支持 JSON 导入/导出备份

## AI 接口

在应用左下角打开"AI 接口设置"，默认会填入 DeepSeek 的 OpenAI 兼容接口：

- 接口地址：`https://api.deepseek.com/chat/completions`
- 模型名：`deepseek-v4-flash`
- API Key：你的 DeepSeek API Key
- 分类提示词

当前 demo 会通过本地 `server.js` 的 `/api/chat-completion` 代理转发到 DeepSeek，避免浏览器直连第三方 API 时遇到跨域限制。正式桌面版建议对 API Key 做系统级安全存储。
